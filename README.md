### Soal Sulit No 1
###### Alur kerja program :
###### 1. Program menanyakan kepada pengguna untuk memasukkan jumlah siswa yang akan direkap nilainya.
###### 2. Program kemudian akan meminta pengguna untuk memasukkan nilai ujian untuk setiap siswa satu per satu menggunakan perulangan for.
###### 3. Setiap nilai yang dimasukkan akan ditambahkan ke total nilai menggunakan rumus:
      totalNilai += nilai;
###### 4. Setelah semua nilai siswa dimasukkan, program akan menghitung nilai rata-rata menggunakan rumus:
    rataRata = totalNilai / jumlahSiswa;

### Soal Sulit No 2
##### Alur kerja program :
##### 1. Program menanyakan kepada pengguna untuk memasukkan jumlah baris dan jumlah kolom dari matriks yang akan dijumlahkan.
##### 2. Program akan menyiapkan dua buah array 2 dimensi (Matriks A dan Matriks B) berdasarkan ukuran yang dimasukkan pengguna.
##### 3. Pengguna diminta untuk mengisi setiap elemen dari Matriks A, kemudian mengisi elemen dari Matriks B.
##### 4. Setelah kedua matriks terisi, program akan melakukan proses penjumlahan menggunakan dua perulangan for, dengan rumus:
    hasil[i][j] = matriksA[i][j] + matriksB[i][j];
